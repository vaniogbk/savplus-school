# 🎓 Formation Maintenance Informatique & Réseau
### SAVOIR PLUS SCHOOL

Site vitrine de la formation professionnelle en Maintenance Informatique & Réseau certifiante ICDL.

## 🚀 Stack technique
- HTML5 sémantique
- TailwindCSS (CDN)
- Google Fonts (Syne + DM Sans)
- CSS animations & Intersection Observer

## 📋 Sections
- Header fixe avec menu responsive
- Bannière Hero avec stats
- Objectifs de formation (grille 3 colonnes)
- Programme (Maintenance + Réseaux)
- Prérequis d'entrée
- Débouchés professionnels
- Certification ICDL (badges)
- Caractéristiques (tableau)
- Public cible
- À propos
- Footer Contacts + formulaire rapide

## 🔄 CI/CD
Déploiement automatique sur **Vercel** et **Netlify** à chaque push sur `main` via GitHub Actions.

## 📦 Déploiement
- GitHub: `formation-maintenance-reseau`
- Vercel: `formation-maintenance-reseau.vercel.app`
- Netlify: `formation-maintenance-reseau.netlify.app`
